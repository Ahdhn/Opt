**CERES vs Opt** Analyzing CERES =============================================================================== The following uses notation from: https://en.wikipedia.org/wiki/Conjugate_gradient_method CERES directly modifies the Jacobian by multiplying it with a scaling matrix S. S is a diagonal matrix where each entry $$s_i = \frac{1}{1+\sqrt{(J^tJ)_{ii}}}$$ We can write this without resorting to entry-wise notation if we let $$Q = \mbox{diag}(J^tJ)$$ and $U$ be the matrix such that $$Q = U^tU$$ Then $S$ is just $$S = (U + I)^{-1}$$
